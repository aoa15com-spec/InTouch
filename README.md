🛰️ InTouch Messenger Pro
A Modern, Real-time Communication Suite

InTouch Messenger Pro is a sleek, single-page communication platform designed for seamless user interaction. It combines a vibrant, "Outfit"-inspired aesthetic with powerful messaging and video calling capabilities.

✨ Key Features
Unique 6-Digit ID System: Every user is assigned a permanent, non-repeating 6-digit identification code stored via localStorage for persistent identity.

Rich Profile Customization: Integrated profile setup allowing users to upload avatars, set bios, and manage their digital identity.

Smart Messaging: A dynamic chat interface that supports real-time text exchange and automated link detection.

High-Fidelity Video Calls: Built-in WebRTC support for peer-to-peer video conferencing with native camera and microphone integration.

Responsive Fluid Design: A fully adaptive UI that scales from desktop sidebar views to mobile-friendly layouts using modern CSS Grid and Flexbox.

🛠️ Technical Stack
Frontend: HTML5, CSS3 (Custom Variables & Keyframe Animations), JavaScript (ES6+).

Real-time Engine: Socket.io for signaling and instant message delivery.

Protocol: WebRTC for high-performance, low-latency video streaming.

Storage: Browser localStorage for session persistence and credential caching.

🚀 Getting Started
Clone the Repository: git clone https://github.com/your-username/intouch-messenger.git

Open locally: Simply open index.html in any modern web browser.

Deployment: This project is pre-configured for one-click deployment via GitHub Pages or Netlify.

🔒 Security & Privacy
InTouch uses secure browser APIs for media access. Camera and microphone permissions are requested only during active calls and are never recorded or stored on a server.
